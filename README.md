# Ikenna CRM/ERP

This repository contains the Ikenna CRM & ERP platform (in development).

## Feature flags
- Raptor mini (Preview) is enabled by default for all clients. To disable it, set the environment variable to false in `.env`:

```env
NEXT_PUBLIC_ENABLE_RAPTOR_MINI=false
```

## Local setup
1. Copy `.env.example` to `.env` and edit values.
2. Run `npm install`
3. Run `npm run dev`
