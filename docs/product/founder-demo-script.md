# Founder Demo Script

Length: 7 minutes
Date: 2026-06-07

## Demo Flow
1. Landing page
   - Position Civis as one calm operating system for CRM, finance, HR, projects, and operations.
   - Point out honest pricing: sign-up is live, public checkout is intentionally not presented as live.
2. Login
   - Enter through the real app, not a mocked preview.
3. Dashboard / Civis Pulse
   - Show Today’s priorities, Setup blockers, Trust & security, and Suggested next actions.
   - Emphasize that Civis Pulse is grounded in real queues and launch blockers, not fake “AI insight.”
4. CRM flow
   - Open CRM.
   - Show contacts, companies, and deals tabs.
   - Mention that CRUD is meant to persist and that the pipeline is designed for real sales motion.
5. AI command moment
   - Use: “take me to sales pipeline”
   - Use: “what should I do next?”
   - Explain that navigation is deterministic and permission-aware before any provider-backed generation.
6. Accounting approval flow
   - Show invoice/expense approval request and explain persisted approval lifecycle.
7. Operations approvals
   - Show approvals queue and org-scoped review posture.
8. Admin trust / org isolation
   - Show that founder-only surfaces are separated from workspace admin experience.
   - Mention route guards, org isolation, and role-based access.
9. Marketing honesty
   - Show that marketing is preview-only rather than pretending to be production-ready.
10. Pricing / sign-up close
   - Return to pricing or sign-up and frame the rollout path honestly.

## What Not To Demo Yet
- Public self-serve billing checkout
- Live campaign sending from Marketing
- Any provider-backed AI flow you have not validated live
- Any upload/provider path you have not validated after redeploy
- Any flow still blocked by backup / restore / fake-data evidence gaps

## Known Blocked Items
- Provider-backed flows that still need live evidence
- Backup / restore drill evidence
- Fake-data review evidence
- Some non-critical settings persistence still intentionally unsupported

## Fallback Talking Points
- “We disabled or labelled anything that would create fake confidence.”
- “The product is being launched with honesty: real flows are real, blocked flows are visible.”
- “We chose deterministic AI routing and grounded summaries over flashy but unreliable behaviour.”

## Investor / Customer Positioning
- For investors: Civis is moving from template breadth to a coherent operator-first system with credible access control and launch discipline.
- For customers: Civis gives one workspace for revenue, money, people, and ops without forcing an enterprise-only learning curve.
