# Civis Product Overview

Updated: 2026-03-17

## What Civis Is

Civis is a CRM/ERP platform for running business operations from one workspace. It combines customer management, finance, HR, projects, inventory, client communication, reporting, and AI assistance inside a single product.

The product is being shaped around practical business execution rather than isolated tools. The goal is to reduce the number of disconnected systems a team needs to manage daily work.

## Core Problem Civis Solves

Most small and mid-sized teams split work across:

- CRM tools for sales
- accounting tools for invoices and expenses
- HR spreadsheets for employee tracking and payroll
- project tools for execution
- storage tools for client sharing
- messaging tools for reminders and follow-ups

That creates duplicated data, broken context, and poor operational visibility.

Civis is designed to centralize that operational flow.

## Current Product Direction

The platform is moving toward a business operating system with:

- live operational visibility
- role-aware dashboards
- AI-assisted execution
- integrated documentation and reporting
- client-facing collaboration surface

This is not positioned as a clone of a single existing product. It is closer to an operations layer that connects CRM, finance, people, delivery, and communication.

The product is beyond mockup stage. It already includes:

- a real Next.js application
- Prisma/Postgres data model
- authenticated dashboard
- API layer
- OpenAPI documentation
- security documentation
- AI provider integration surface

Some modules are already committed and deployed, while some backend conversions remain in local progress and still need final migration/deployment.
